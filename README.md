# maple
monstrat on maple

각 셀은 성공이면 1, 실패면 f, 찬스타임의 경우 c로, 파괴는 d로 표기됩니다.

trial3을 예로 들겠습니다.

10성에서 찬스타임을 받아 11성으로, 11성에서 실패해 10성으로,
10성에서 한 번에 성공해 11성으로, 11성에서 실패해 10성으로,
10성에서 2번만에 성공해 11성으로, 11성에서 실패해 10성으로,
10성에서 한 번에 성공해 11성으로, 다시 성공해 12성으로, 다시 성공해 13성으로,
13성에서 실패해 12성으로, ...

이런 식입니다.
